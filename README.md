## ✨ Features

- 🎨 **Premium luxury UI** — Deep Maroon, Royal Gold & Warm Ivory palette
- 📱 **Fully mobile responsive** — tested from 320px to 1440px+
- 🛒 **Full ecommerce flow** — Browse → Product → Cart → Checkout
- 💬 **WhatsApp ordering** — Direct WhatsApp checkout integration
- 🔍 **SEO optimised** — JSON-LD structured data, meta tags, semantic HTML
- ♿ **Accessible** — ARIA labels, keyboard navigation, focus management
- ⚡ **Fast** — Pure HTML/CSS/JS, no framework, ~0 dependencies
- 🛡️ **Admin Panel** — Demo admin dashboard at `/admin/`
