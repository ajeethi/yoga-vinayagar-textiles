🔥 Connect Firebase — Go LIVE in 5 Minutes

Firebase is Google's free database. Once connected, every order placed by a customer will instantly appear here in real-time. It's free for small businesses.

Step 1 — Create Firebase Project
  1. Go to console.firebase.google.com
  2. Click "Add project" → Name it yoga-vinayagar
  3. Disable Google Analytics (optional) → Click Create project
Step 2 — Get Your Config Keys
  1. In your project → Click the Web icon (</>)
  2. Register app name: yoga-vinayagar-web
  3. Copy the firebaseConfig object shown
Step 3 — Enable Firestore Database
  1. In Firebase → Left menu → Firestore Database
  2. Click Create database → Choose Test mode → Select Asia region
  3. Click Enable
